Политика конфиденциальности

Последнее обновление: Июнь 2026 г.

Мы заботимся о вашей конфиденциальности. Настоящая Политика конфиденциальности объясняет, как приложение Nepsis: screen time control (далее — «Приложение») обрабатывает данные пользователей.

1. Локальная архитектура (Local-First)

Приложение спроектировано по принципу «local-first». Это означает, что:

  • Мы не создаем учетные записи пользователей (нет регистрации по имени, номеру телефона или email).
  • Мы не собираем, не храним и не передаем ваши личные данные на собственные серверы.
  • Все данные, связанные с использованием Screen Time API (время работы приложений, блокировки и ограничения), обрабатываются исключительно локально на вашем устройстве под управлением операционной системы iOS. У разработчика Приложения нет технической возможности получить доступ к этой информации.

2. Использование Screen Time API (FamilyControls)

Приложение использует функции Apple Screen Time API для мониторинга и ограничения времени использования других приложений. Все разрешения запрашиваются стандартными системными диалогами iOS. Данные этого API защищены встроенными механизмами Apple, не покидают ваше устройство и не используются в рекламных целях.

3. Сторонние сервисы и обработка данных подписок

Для предоставления и управления встроенными покупками и подписками Приложение использует сторонний сервис RevenueCat (RevenueCat, Inc.).

RevenueCat собирает и обрабатывает следующие обезличенные данные:

  • Идентификаторы (Identifiers): Случайный анонимный буквенно-цифровой идентификатор (User ID), автоматически генерируемый для вашего устройства, чтобы проверять статус вашей подписки.
  • История покупок (Purchase History): Информация о транзакциях (дата покупки, тип подписки, статус продления), обрабатываемая через зашифрованные чеки Apple (Apple purchase receipts).

Эти данные передаются в RevenueCat в зашифрованном и строго анонимном виде. Они необходимы исключительно для обеспечения функциональности приложения (проверки наличия Premium-доступа) и внутренней аналитики продаж. Эти данные не связаны с вашей личностью и не используются для трекинга в рекламных сетях.

Вы можете ознакомиться с политикой конфиденциальности сервиса на официальном сайте: RevenueCat Privacy Policy.

4. Изменения в Политике конфиденциальности

Мы можем периодически обновлять эту Политику конфиденциальности. Изменения вступают в силу с момента их публикации на этой странице. Мы рекомендуем регулярно проверять её актуальность.
